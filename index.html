It is a website that acts as a “mini” wiki for the game Left 4 dead 2. It features information regarding the game, including the maps, characters, and its weapon stats. The data used is based off of the actual wiki made for the left 4 dead 2. [Link to the website]

CondiPlant

CondiPlant is a mobile application that uses machine learning and image recognition to identify and classify root crop diseases within Cabuyao City. Upon classification, the app displays the top three identified disease and a short description on each of them can be viewed later on. These includes Mitigation Steps. [Link to the project]
